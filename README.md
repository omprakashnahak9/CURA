## Getting Started

### Prerequisites
- Android Studio Hedgehog or later
- JDK 17 or higher
- Android SDK 34

### Installation

1. Clone the repository
```bash
git clone https://github.com/omprakashnahak9/CURA.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or physical device
